Folder Tree

This shows the first level of the folder tree with deeper levels for selected items. The select folder is shown as an open folder icon.

Images List

The list shows all of the images in the selected folder and all of its subfolders. Images can be selected by Status , File type, ... [ToDo].

The thumbnail image is a link to a modal dialog which displays the image full-size or the width of the dialog, whichever is smaller.
