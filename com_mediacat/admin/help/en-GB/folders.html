Media Cat Folders

This page shows the list of folders in use for storing either Images or Files. Select from the drop-down list to switch. The list is displayed on the left of the screen, or near the top on narrow screens.

Select a Folder Tree item to perform one of the following actions:

Indexer
This action processes all of the files in the selected folder and inserts or updates a record in the database. It may take quite a long time. Perhaps several seconds for 1000 images.
Hasher
This action processes all of the files in the selected folder and updates the file hash field in the database record. It may take quite a long time. Perhaps several seconds for 1000 images.
Create New Folder
This action prompts for a new folder name and creates the folder. All characters except alphanumers and minus (-) are replace by - (minus). You will be asked to confirm that the file name is acceptable. Please use minus characters to separate words.
Trash
This action ... moves the file to a trash folder from where it can be retrieved or deleted as necessary. Currently not implemented.
